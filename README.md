Here is the quick-reference list for Windows.

Keybind	Action
Ctrl + C	copy line or selection
Ctrl + V	paste
Ctrl + Z	undo
Ctrl + Shift + Z	redo
Ctrl + /	toggle comment for line or selection
Tab	indent line or selection
Shift + Tab	unindent line or selection
Alt + Up	move line or selection up
Alt + Down	move line or selection down
Alt + Left	jump left to previous word
Alt + Right	jump right to next word
Home	jump to start of line
End	jump to end of line
Ctrl + Home	jump to start of document
Ctrl + End	jump to end of document
Ctrl + L	select current line
Shift + Up	select upward
Shift + Down	select downward
Shift + Left	select leftward
Shift + Right	select rightward
Ctrl + Left-Click	place additional cursor
Ctrl + D	select word at cursor
 (subsequent presses of D will select next instance of same word)
Windows + Alt + S	auto-format file
